# Concurrent State Management - Learning Path Phases 1-8

# Getting Started with Phase 1

When you're ready to begin, start with Phase 1. Build the single-threaded baseline, instrument it heavily, and run all the experiments. The measurements you gather will motivate every subsequent phase.

**First concrete task:** Implement a message generator that yields messages at a configurable rate. Make it work, then make it measurable.
